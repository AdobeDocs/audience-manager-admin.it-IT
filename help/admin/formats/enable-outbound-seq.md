# Abilita trasferimenti di file sequenza Hadoop per outbound {#enable-hadoop-sequence-file-transfers-for-outbound}

Potete abilitare i trasferimenti in batch binari [!DNL Hadoop Sequence Files] ([!DNL SEQ]) per i clienti di Audience Manager.

<!-- REMOVED FROM PUBLIC DOCS: The advantages of using [!DNL Hadoop SEQ] files are listed in the [public documentation](https://marketing.adobe.com/resources/help/en_US/aam/outbound-seq-files.html). -->

Per abilitare [!DNL SEQ] i trasferimenti di file in uscita:

1. Selezionate la società e la destinazione per la quale desiderate abilitare [!DNL SEQ] i trasferimenti di file in uscita.
1. Nell' **[!UICONTROL Edit Destinations]** interfaccia, scorri verso il basso fino **[!UICONTROL Batch Data]** a.
1. Seleziona protocollo: **[!UICONTROL FTP]** oppure **[!UICONTROL S3]**.
1. Selezionate Formato: **[!UICONTROL AAM-Outbound-SEQ (415)]**.
1. Confermate facendo clic su **[!UICONTROL Save]** nella parte inferiore della pagina. Il cliente ora può inviare file di sequenze Hadoop alla destinazione.
